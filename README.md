## AWS + Snowflake + Tableau Project

This project demonstrates an end-to-end cloud-based data analytics pipeline using AWS S3, Snowflake, and Tableau.

---

### 📊 Objective
To analyze usage patterns and cost savings using a scalable cloud data pipeline and interactive dashboard.

---

### ☁️ AWS (S3)
- Created Amazon S3 bucket
- Uploaded dataset
- Configured IAM roles for secure integration

---

### ❄️ Snowflake
- Created integration with AWS S3
- Loaded data into Snowflake tables
- Performed data transformation using SQL
- Prepared dataset for reporting

---

### 📊 Tableau Development
- Connected Tableau to Snowflake
- Built visualizations for:
  - Monthly Usage Consumption
  - Cost Savings Analysis

---

### 📈 Dashboard
- Designed interactive dashboard
- Combined multiple insights into a single view
- Focused on usage trends and cost optimization

---

### 🚀 Deployment
- Published workbook to Tableau Cloud
- Enabled online access and sharing

---

### ✅ Outcome
- Built scalable cloud analytics solution
- Provided insights into usage and cost optimization

---

### 🛠 Tools Used
- AWS S3
- Snowflake
- Tableau Desktop
- Tableau Cloud
- SQL (Snowflake SQL)
